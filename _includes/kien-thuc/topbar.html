{%- comment -%} The sub-brand lockup: Alpha's logo + "Kiến thức" — ONE mark, not a logo next to a word. "Kiến thức" is set in type but must behave like part of the logo: it never wraps, never underlines, never gets link styling. It is perceptually centered on the whole logo, sized so its caps match the "alpha" wordmark's x-height (the lockup geometry lives in _sass/kien-thuc/_theme.scss). Kiến thức is a branch of Alpha, not a separate company. The lockup always links to the hub (/kien-thuc/), so it doubles as the way back from any article.
