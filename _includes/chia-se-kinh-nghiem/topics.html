{%- comment -%} Every topic in the library, on one line, as links to the topic pages (/chia-se-kinh-nghiem/chu-de//). Sits above the first post on the hub and on every topic page — the one place the reader can see the whole map. Ordered by how many posts carry the topic, most first: the strip should open on what this library is mostly about, and it re-orders itself as the archive grows. These are indigo, not metadata grey: unlike the tag on a feed card (which labels the post it sits under), this row exists ONLY to be clicked (§3 — indigo means actionable). Liquid has no "sort an array of objects by a computed number", so each topic is flattened to a sortable string — "" — sorted, and reversed. `count` is offset by 1000 so every row is the same width and a plain string sort orders it numerically; the declaration index breaks ties in favour of the order the data file lists them in. {%- endcomment -%} {%- assign kt_rows = "" | split: "" -%} {%- for kt_pair in site.data.chia_se_kinh_nghiem.topics -%} {%- assign kt_count = site.posts | where: "topic", kt_pair[0] | size -%} {%- if kt_count > 0 -%} {%- assign kt_rank = kt_count | plus: 1000 -%} {%- assign kt_tie = 999 | minus: forloop.index -%} {%- capture kt_row -%}{{ kt_rank }}|{{ kt_tie }}|{{ kt_pair[0] }}{%- endcapture -%} {%- assign kt_row = kt_row | split: "\n" -%} {%- assign kt_rows = kt_rows | concat: kt_row -%} {%- endif -%} {%- endfor -%} {%- assign kt_rows = kt_rows | sort | reverse -%}
