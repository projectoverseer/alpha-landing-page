# content-archive

Raw source material for site content. **Not part of the Jekyll build**
(excluded in `_config.yml`).

## facebook-posts/

The founder's Facebook posts, copied as plain text, one file per post, named
`YYYY-MM-DD-<slug>.txt` — the same date + slug as the published version in
`_posts/`, so source and published article pair up 1:1.

These are the unedited originals (including the Facebook-only lines: CTA
footers, "comment nhé" asks, attachment references). The published Markdown in
`_posts/` is the edited version: converted headings, those lines removed,
editorial CTAs inserted, image placeholders marked. When in doubt about what
the author originally wrote, this folder is the source of truth.
